>project data:
  PROJ.txt data format:
    ID: "####"
    TAG: "tag:lionkimbro@gmail.com,2022-03-21:example" (Tag URIs -- taguri.org)
    TITLE: "Human Readable Title"
    CREATED: "yyyy-mm-dd"
    TAGS: ["foo", "bar", "baz", ...]
    LOC: "0000"  -- the subdirectory that contains the project resources
    HOOK: "human readable hook string"
  update notice:
    variable: g[UPDATE]
      when ACTIVE: in the last cycle, the project data was updated
      when TODO: an ACTIVE cycle is pending
      when None: no updates
      permission: anybody can set this to TODO at any time, and that is safe
        caveat: if set to TODO while an ACTIVE is processing, there may be a double-hit for some
        important: operations from the update should be idempotent
